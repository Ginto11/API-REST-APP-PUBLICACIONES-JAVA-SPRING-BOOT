Acerca de

Este proyecto fue realizado con el fin de probar como se hace una API REST en java usando SPRING BOOT, en la cual se pueden crear publicaciones y cada una tiene su autor, descripcion y etiquetas.

Para el uso de repositorios se utilizo Spring Data JPA, se utilizo Hibernate para el mapeo de entidades a la base de datos. Como gestor de bases de datos se utilizo MySQL
